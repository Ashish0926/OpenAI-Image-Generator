
# OpenAI-Image-Generator


A tool to convert text to images using OpenAI's ImageGPT3 API.




## Run Locally

Clone the project

```bash
  git clone https://github.com/Ashish0926/OpenAI-Image-Generator.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
